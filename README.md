## Genius Car Service (MERN Stack)

### Description

Genius Car Service is a full-stack web application for managing car service bookings, orders, and service management. It features authentication, protected routes, and a modern UI for both customers and service providers.

### Tech Stack

- **Frontend:** React, React Router, Axios
- **Authentication:** Firebase Auth
- **Backend:** Node.js, Express.js, MongoDB (API not included in this repo)
- **Styling:** CSS Modules, Custom CSS

### About

This project demonstrates a typical MERN stack application with user authentication, CRUD operations for services, and order management. It is designed for learning and portfolio purposes.
